How to design a favicon using favicon.io

Following Steps are used to create a favion.

  • Open favicon.io site.
  • Select Favicon Generators Style.
  • PNG → ICO
  • TEXT → ICO
  • 😀 → ICO
  • After that we have to select one.
  • In that particular style we have to give Text, Backgroud, Font Family, Font Variant, Font Size, Font Color, Background Color.
  • Now we have to download the favicon zip file.
  • After that we have to extract that file and move favicon file to exact folder of that code.
